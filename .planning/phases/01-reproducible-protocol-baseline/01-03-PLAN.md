# Phase 1 Plan 03: Single-VM Remediation

**Status:** Complete
**Added:** 2026-07-14
**Supersedes:** the external VM boundary in plans 01-01 and 01-02

## Objective

Remove the second interpreter and runtime value model. Make the workspace
`neo-vm` crate the only Neo N3 VM authority and use `StackItem` directly across
all consensus-facing projections, storage codecs, and execution results.

## Tasks

- [x] Remove the alternate interpreter from `ApplicationEngine` canonical execution.
- [x] Port required opcode, instruction, limits, exception, ordered-map,
  script-validation, stack-item-type, VM-state, and syscall metadata into
  `neo-vm/src/vm_types/` with license provenance.
- [x] Remove the external VM dependency from root, workspace crates, fuzzing,
  lockfiles, dependency policy, and build tooling.
- [x] Delete the graph conversion layer and make `Interoperable` operate on
  `StackItem` directly.
- [x] Migrate serialization, payload, manifest, execution, native-contract,
  blockchain, node-probe, and server-side RPC paths to direct `StackItem` APIs.
- [x] Keep remote RPC values in an immutable `RpcStackItem` DTO that cannot be
  promoted implicitly into executable VM state.
- [x] Preserve pre-Domovoi notification identity and Domovoi deep-copy alias
  semantics directly on the local object graph.
- [x] Update ADRs and maintained architecture, operations, getting-started,
  roadmap, state, requirements, and dependency guards.
- [x] Run focused crate tests, dependency-policy tests, and the workspace
  all-target build; retain results in `01-VM-CUTOVER-EVIDENCE.md`.

## Acceptance

1. `cargo metadata` contains the local `neo-vm` package and no external VM
   package or Git VM source.
2. Active Rust source contains no legacy runtime value type or conversion API.
3. `ApplicationEngine` has one interpreter and one execution-limit authority.
4. Direct `StackItem` tests cover numeric boundaries, compound identity,
   notifications, storage codecs, RPC rendering, and database decoding.
5. The workspace all-target build and focused owner-crate suites pass.
